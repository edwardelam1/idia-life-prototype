# IDIA Life Smart Contracts

The IDIA governance contracts are deployed on **Base Mainnet** (Chain ID 8453).
Live addresses are tracked in `src/config/contracts.ts` under the `mainnet`
deployment block — that file is the on-chain source of truth.

The legacy `IDIATokenBeta` non-transferable receipt contract has been retired
and removed from this repository.

For deploy or admin operations against the live protocol, work from the
governance multisig and the verified contracts on BaseScan.
